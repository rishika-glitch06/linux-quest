# 🧪 Lab 02 - Explore the Linux File System

## 🎯 Objective

Understand the Linux directory structure and practice navigating it.

---

## Tasks

Run these commands:

```bash
pwd
cd /
ls
cd /home
pwd
cd ~
pwd
ls
```

---

## Questions

1. Which directory is at the top of the Linux file system?
2. What does `~` represent?
3. What is the difference between `/` and `~`?
4. Which directory stores system configuration files?
5. Which directory contains temporary files?

---

## Expected Learning

By the end of this lab, you should be comfortable navigating the Linux directory structure and understanding the purpose of common system directories.
