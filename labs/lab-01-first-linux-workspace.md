# 🧪 Lab 01 - Create Your First Linux Workspace

## 🎯 Objective

Practice the most common Linux navigation and file creation commands.

---

## Tasks

Run the following commands one by one:

```bash
pwd
ls
mkdir LinuxQuest
cd LinuxQuest
touch README.md
touch notes.txt
ls
pwd
cd ..
rmdir TestFolder
```

---

## Expected Learning

By completing this lab, you should be able to:

- Navigate directories
- Create folders
- Create files
- View directory contents
- Understand your current location
