# 🧪 Lab 03 – File Management Practice

## 🎯 Objective

Practice creating, copying, moving, finding, and deleting files.

---

## Tasks

Run the following commands:

```bash
mkdir LinuxLab
cd LinuxLab

touch notes.txt
cp notes.txt backup.txt

mkdir Archive
mv backup.txt Archive/

find . -name "*.txt"

file notes.txt

rm notes.txt
```

---

## Questions

1. Which command copied the file?
2. Which command moved the backup file?
3. Which command searched for `.txt` files?
4. Which command identified the file type?
5. Which command deleted the original file?

---

## Expected Learning

After completing this lab, you should be able to:

- Create files and folders
- Copy files
- Move and rename files
- Search for files
- Identify file types
- Delete files safely
