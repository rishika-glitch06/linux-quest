# 📚 Linux Basics

## 1. What is Linux?

**Answer:**
Linux is an open-source, Unix-like operating system kernel created by Linus Torvalds in 1991. Linux distributions combine the kernel with software to create a complete operating system.

---

## 2. Why is Linux popular?

**Answer:**

- Open Source
- Secure
- Stable
- Fast
- Highly customizable
- Widely used in servers, cloud computing, DevOps, AI, and cybersecurity

---

## 3. What is the Linux Kernel?

**Answer:**

The kernel is the core of the operating system. It manages hardware, memory, processes, devices, and communication between software and hardware.

---

## 4. What is a Linux Distribution?

**Answer:**

A Linux distribution is a complete operating system built around the Linux kernel.

Examples:

- Ubuntu
- Fedora
- Debian
- Arch Linux

---

# 🖥️ Terminal Basics

## 5. What is the difference between GUI and CLI?

**GUI**
- Uses windows, icons, and menus.

**CLI**
- Uses text commands.
- Faster and more powerful for advanced tasks.

---

## 6. What does the `pwd` command do?

Displays the current working directory.

---

## 7. What does the `ls` command do?

Lists files and directories.

---

## 8. What is the purpose of the `cd` command?

Changes the current working directory.

---

# 📂 Linux File System

## 9. What is the Root Directory?

The Root Directory (`/`) is the top-most directory in Linux.

---

## 10. What does `~` represent?

The current user's Home Directory.

---

## 11. Difference between Absolute and Relative Path?

**Absolute Path**
- Starts with `/`
- Always points to the same location.

**Relative Path**
- Starts from the current directory.

---

# 📁 File Management

## 12. What does `cp` do?

Copies files or directories.

---

## 13. What does `mv` do?

Moves or renames files and directories.

---

## 14. What does `rm` do?

Deletes files. With `-r`, it can remove directories and their contents.

---

## 15. Difference between `rm` and `rmdir`?

- `rm` removes files and directories.
- `rmdir` removes only empty directories.

---

## 16. What does `find` do?

Searches for files and directories.

---

## 17. What does the `file` command do?

Identifies the type of a file.
